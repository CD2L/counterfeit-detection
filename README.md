# counterfeit-detection demo

## Information
This demo is using the code of CD2L organization. CD2L codes belongs to Centrale Lille. 

## Usage
To run the demo you'll have to install [flask](https://flask.palletsprojects.com/en/2.2.x/installation/).
```bash
pip install Flask
```
Then, you can use directly ```flask run``` in the root directory to run the demo.

Don't push image or binary file on this repo
